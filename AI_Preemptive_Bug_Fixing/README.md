# AI: Preemptive Bug Fixing

## Description
This task focuses on using AI to identify and fix logical and memory safety issues in a C linked list function.

## Files
- initial_code.c : Contains the vulnerable function
- refactored_code.c : Contains the corrected function
- README.md : Task documentation

## AI Tool Used
Claude AI

## Issues Identified
- Logical error in linking the new node
- Missing NULL check after malloc
- Potential segmentation fault
